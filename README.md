## Getting Started
### Prerequisites
<ul>
    <li>
        <a href="https://nodejs.org/en">Node.js 18.17.1</a>
    </li>
</ul>

### Usage
1. Open the IS442-FRONTEND folder
2. Install required dependencies

```
npm i
```
3. Start frontend

```
npm run serve
```
